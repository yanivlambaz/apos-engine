# APOS - SEO SYSTEM

Version: 1.0

Status: STABLE

Classification: CORE KERNEL

---

# Purpose

This document defines every SEO rule used by APOS.

SEO is not an optimization step.

SEO is part of the architecture.

Every generated page must satisfy these rules before release.

---

# SEO Objectives

Every page must

Be understandable by search engines.

Answer a clear search intent.

Strengthen topical authority.

Support AI systems.

Support internal navigation.

Generate qualified traffic.

---

# Search Intent

Every page serves exactly one primary intent.

Allowed intents

Informational

Commercial Investigation

Transactional

Navigational

Do not mix primary intents.

---

# URL Rules

URLs must be

Readable

Short

Stable

Semantic

Examples

/services/private-nurse

/guides/private-nursing

/articles/private-nurse-cost

Never use

/page1

/post-123

/final-page

---

# Metadata

Every page requires

SEO Title

Meta Description

Canonical URL

Robots Directive

Open Graph

Twitter Card

Language

Last Updated

Author (where applicable)

---

# Titles

Requirements

Exactly one H1

Unique Title

45–60 characters preferred

Primary keyword appears naturally

Readable by humans

No keyword stuffing

---

# Meta Description

Requirements

140–160 characters preferred

Summarize page purpose

Include value proposition

Encourage click naturally

Unique for every page

---

# Heading Structure

Exactly one H1

H2 for major sections

H3 for subsections

Never skip hierarchy

Avoid empty headings

---

# Internal Linking

Every page must link to

Parent

Children

Related Topics

Related Services

Relevant Articles

Relevant Guides

Relevant FAQ

Homepage

No orphan pages.

---

# External Links

External links should

Support credibility

Reference authoritative sources

Open in new tab when appropriate

Never overload pages with external links

---

# Images

Every image requires

Meaningful filename

ALT text

Caption when useful

Responsive sizes

Lazy loading

Optimized format
