# Purpose

The Homepage is not a marketing page.

It is the central authority page of the entire platform.

Every important entity inside the platform should be discoverable directly or indirectly from the Homepage.

The Homepage has four primary objectives:

1. Build trust immediately.
2. Explain the business clearly.
3. Direct users toward the correct journey.
4. Distribute authority throughout the platform.

Everything else is secondary.
