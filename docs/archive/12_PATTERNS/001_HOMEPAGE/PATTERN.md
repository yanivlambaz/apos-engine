# Purpose

This Pattern defines the complete implementation standard for every Homepage built using APOS.

The Homepage is the central authority page of the platform.

It is not a marketing page.

It is the primary trust-building page.

---

# Business Goals

The Homepage must accomplish all of the following.

✓ Build trust

✓ Explain the business

✓ Demonstrate expertise

✓ Support SEO

✓ Support AI

✓ Generate qualified leads

✓ Guide users

✓ Distribute authority

Failure to satisfy any objective means the Homepage is incomplete.

# Hero Rules

Exactly one H1.

Maximum two CTAs.

Professional image.

Clear value proposition.

Trust statement.

No sliders.

No videos that autoplay.

No rotating banners.

---

# CTA Rules

One primary CTA.

One optional secondary CTA.

Examples:

Call

WhatsApp

Contact Form

Appointment

Never more than one dominant action.

---
